# Health Feature

This module centralizes the operational health dashboard, including API hooks, UI components, and helper utilities for working with infrastructure diagnostics.

- **components/**: Presentation components such as the health status cards and diagnostic forms.
- **hooks/**: Data fetching hooks that orchestrate health checks.
- **services/**: API clients and integration helpers.
- **utils/**: Pure helper functions for shaping health data.

All exports are re-exported from `index.js` for convenient consumption.
