# User Statistics

At the top of the User Management dashboard, you'll find a series of statistical cards. These cards provide a quick, at-a-glance summary of the user base according to their current status.

## Stat Cards

-   **Total Users:**
    -   Displays the total number of user accounts that exist in the system.

-   **Active:**
    -   Shows the number of users who have signed in within the last 30 days.

-   **Pending Verification:**
    -   Shows the number of users who have registered but have not yet completed all onboarding steps (e.g., email verification).

-   **Invited:**
    -   Displays the count of users who have been sent an invitation to join but have not yet accepted it.

-   **Suspended:**
    -   Shows the number of user accounts that have been temporarily disabled by an administrator.

-   **Verified:**
    -   Displays the total number of users who have successfully verified their email and profile.
