# Admin

The Admin section provides powerful tools for managing the application, its users, and its security. Access to this section is restricted to users with the "Administrator" role.

- **[User Management](./user-management/index.md):** A comprehensive dashboard for managing all users in the system.
- **Access Control:** (Coming soon) Manage roles and permissions.
