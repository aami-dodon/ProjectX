# Governance

The Governance section is the central hub for managing your organization's compliance posture. It provides tools to define, monitor, and review your internal controls and the automated checks that validate them.

Access to this section is available to users with the "Admin" and "Compliance Officer" roles.

## Features

This section is organized into several key pages:

-   **[Overview](./overview.md):** A high-level dashboard summarizing the overall state of your governance activities.
-   **[Control Catalog](./control-catalog.md):** A comprehensive library of all the compliance controls defined in your organization.
-   **[Control Scoreboard](./control-scoreboard.md):** A dashboard that visualizes the performance and effectiveness of your controls.
-   **[Check Catalog](./check-catalog.md):** A library of all the automated checks used to gather evidence for your controls.
-   **[Result Explorer](./result-explorer.md):** A tool for exploring the results generated by the automated checks.
-   **[Review Queue](./review-queue.md):** A dedicated workspace for reviewing and actioning check results.
