# User Guide

Welcome to the user guide for the AI Governance Platform. This guide provides detailed, step-by-step instructions for every feature and workflow in the application.

This guide is organized into sections that mirror the structure of the application. Use the sidebar to navigate to the specific feature you want to learn about.

## Sections

- **[Home](./01-home/index.md):** Learn about the main dashboard and its components.
- **[Account](./02-account/index.md):** Find out how to manage your personal account settings.
- **[Admin](./03-admin/index.md):** For administrators, this section covers user management, access control, and system health.
- **[Probes](./04-probes/index.md):** Learn how to manage and monitor probes.
- **[Governance](./05-governance/index.md):** Understand how to manage compliance controls and checks.
- **[Frameworks](./06-frameworks/index.md):** See how to work with compliance frameworks.
- **[Evidence](./07-evidence/index.md):** Learn about managing and uploading evidence.
- **[Tasks](./08-tasks/index.md):** Find out how to view and manage your tasks.
- **[Dashboards](./09-dashboards/index.md):** Explore the various data dashboards available.
