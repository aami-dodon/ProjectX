# Dashboards

The Dashboards section provides a collection of specialized dashboards for visualizing key compliance and governance metrics.

Access to this section is available to users with the "Admin," "Compliance Officer," "Auditor," and "Executive" roles.

## Available Dashboards

-   **[Framework Scores](./framework-scores.md):** A dashboard for viewing the compliance scores of your frameworks.
-   **[Control Health](./control-health.md):** A dashboard for monitoring the health and performance of your controls.
-   **[Remediation Dashboard](./remediation-dashboard.md):** A dashboard for tracking the status of remediation efforts.
-   **[Evidence Coverage](./evidence-coverage.md):** A dashboard for visualizing the extent of your evidence collection.
