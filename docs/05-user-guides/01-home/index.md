# Home

The Home page is the main dashboard of the application. It provides a high-level overview of key metrics and data points, allowing you to quickly assess the status of your operations.

The page is composed of several components:

- **[Section Cards](./section-cards.md):** A set of cards displaying key performance indicators.
- **[Interactive Chart](./interactive-chart.md):** An interactive area chart for visualizing data over time.
- **[Data Table](./data-table.md):** A detailed and interactive table for viewing and managing data.
