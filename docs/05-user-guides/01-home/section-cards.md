# Section Cards

The top of the Home page features a set of four cards that provide a quick, high-level overview of key performance indicators (KPIs).

## KPIs Displayed

1.  **Total Revenue:**
    *   **Value:** Displays the total revenue.
    *   **Trend:** Shows the percentage change over the current period (e.g., "+12.5%").
    *   **Description:** Provides context for the trend, such as "Trending up this month."

2.  **New Customers:**
    *   **Value:** Shows the number of new customers acquired.
    *   **Trend:** Shows the percentage change (e.g., "-20%").
    *   **Description:** Provides context, such as "Acquisition needs attention."

3.  **Active Accounts:**
    *   **Value:** Displays the total number of active user accounts.
    *   **Trend:** Shows the percentage change (e.g., "+12.5%").
    *   **Description:** Provides context, such as "Strong user retention."

4.  **Growth Rate:**
    *   **Value:** Shows the current growth rate as a percentage.
    *   **Trend:** Shows the percentage change (e.g., "+4.5%").
    *   **Description:** Provides context, such as "Steady performance increase."

*Note: The data displayed in these cards is currently representative and will be connected to live data in a future update.*
